Falling Through (Virtual Dive)
======

3D Arcade Game made for Occulus Rift & other VR devices, mobile and tablet (Adnroid + iOS). Avoid obstacles and try get the best score through proceduraly generated levels.
1 world is available and testable

Features
--------

* VR Compatible
* Procedurally generated levels
* Scoring

Platforms
--------

* PC (Oculus VR)
* PC (Keyboard)
* Android (Phone + Tablet)
* iOS (Phone + Tablet)

Usage
-------

Unity3D crossplatform

* Load the project with unity after cloning the repository

Development Quality
-------

Documentation: None - Feel free to make one  
Code quality: Medium - Prototype, prepared for production development

Dependancies & Engines
-------

* Unity3D
